# Cover Letter AI

Cover Letter AI helps job seekers quickly generate tailored cover letters by analyzing a job description and matching it with the content of their CV. Simply upload a PDF CV, paste the job link or description, and get a customized cover letter ready to use.

## Features

- Upload a PDF CV
- Input a job link or paste the job description directly
- Generate a cover letter aligned with your experience
- Copy the result or download it as a PDF
- Responsive interface optimized for desktop and mobile
- Built-in loading spinner and smooth animations

## How It Works

1. Upload your CV in PDF format.
2. Enter a job posting link or paste the job description.
3. Click "Generate" to create a cover letter.
4. Copy or download the result for your application.

## Technologies Used

- HTML, CSS, JavaScript (Vanilla)
- [jsPDF](https://github.com/parallax/jsPDF) for PDF generation
- OpenAI API (server-side) for content generation
- Node.js (for backend API routing)

## Deployment

This app can be deployed on services like DigitalOcean, Vercel, or any VPS that supports Node.js and Apache or Nginx.

To clone and run locally:

```bash
git clone https://github.com/Teerasak-Mairoddee/cover-letter-ai.git
cd cover-letter-ai
npm install
npm run dev 
```
